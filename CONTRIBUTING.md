# ✅ Orientações de contribuições

## ⚡ Como rodar:
### Prepare o ambiente:
> Instale o yarn caso não possua:
> ```shell
> npm install -g yarn
> ```
> Instale as dependências do projeto
> ```shell
> yarn install
> ```

### Front e Back:
> - Inicie o projeto
> ```shell
> yarn dev
> ```

### Testes:
> - Inicie o cypress
> ```shell
> yarn cypress:open
> ```

### Posts:
> - Adicionar um novo post:
> ```shell
> yarn add-post <social-media> <post-url>
> ```
> - Atualizar posts existentes:
> ```shell
> yarn update-posts
> ```
